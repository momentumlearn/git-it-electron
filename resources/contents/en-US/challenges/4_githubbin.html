Working Better, Together

The repository you've created so far is just on your computer, which is great for you, but you still need a way to share your code and work with others. That's what GitHub is for!

GitHub allows people everywhere to upload what they're working on with Git, enabling them to work easily with others on the same project.

Let's get set up to use GitHub for our work!

Add GitHub username to Git

Add the username for your GitHub account to your Git configuration. We'll do this just for the sake of Git-it; it makes it easier to verify the upcoming challenges. Save it exactly as you created it on GitHub and capitalize where capitalized.

For example, if your GitHub username was captainbarnacles, you would set it like this:

git config --global user.username "captainbarnacles"

You can double check what you have set in your Git config by typing:

git config --global user.username

Click the 'Verify' button to confirm that you have successfully configured your GitHub username in Git.

Make sure you type "user.username" above and not "user.name", which would override the name you set in the first challenge and leave you with no username property! If you found you did that, it's ok, just repeat the step in the first challenge to add your name and then complete the step above.

GitHub & Git config usernames do not match

A common error is not having your GitHub username match the case of the one you set with git config. For instance, 'JLord' isn't the same as 'jlord'

To change your username set with Git, just run the same command you did earlier, but with the correct capitalization:

git config --global user.username "yourusername"

When you've made your updates, verify again!
