Practice creating, cloning, and working in project repos at Momentum.

GitHub Classroom

GitHub classroom is a tool that generates a repository on GitHub based on a code project assigned by your instructor.

Clicking on a project link from GitHub classroom will take you to a page that looks something like this:

Once you click the button to "Accept this assignment", GitHub will create a repo just for you, with your GitHub username in its title. When you see the "You're ready to go!" message, click on the link for your repo to be taken to your repo's main page.

Now you can work with this repo just like any other repo on GitHub. Let's walk through the steps that you will do at the start of every new project.

Create the project repo

Usually your instructor will provide you with a link to each project. For this first walkthrough, use the button below.

Clicking this button will take you to a page on GitHub like the one you saw in the previous section. On that page, accept the assignment, then read and follow the instructions you see there.

Once you have a repo on Github, continue to the next step.

Clone the project repo

Click on the green button to "Clone or download" your repo. This will copy the repo's URL to your clipboard.

In your terminal, navigate to the folder where you intend to keep your projects. When you are in the directory you want to be in (check with pwd), type the following command, but in place of url-for-your-repo, paste in the URL you copied from GitHub.

git clone url-for-your-repo

You'll see output indicating the repo is being cloned. When it's done, make sure you cd into the repo.

Make changes, add, commit, and push.

  1. Open the project repo in VS Code.
  2. Make a change in the README file (add or delete something and save the file).
  3. Add changes: git add README.md
  4. Commit changes: git commit -m "Your message here"
  5. Push changes to the main branch: git push origin main

Click the 'Verify' button to confirm that you have pushed changes to your own project repository on GitHub.
